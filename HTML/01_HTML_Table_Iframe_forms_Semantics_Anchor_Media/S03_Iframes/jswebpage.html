What is JavaScript?

JavaScript is a programming language that allows you to create dynamic and interactive web pages. It is the core technology of web development alongside HTML and CSS, and it enables web pages to respond to user actions, control multimedia, and much more.

JavaScript supports various data types, including:

  • string - Represents text (e.g., `'Hello World'`).
  • number - Represents numeric values (e.g., `25`, `3.14`).
  • boolean - Represents `true` or `false`.
  • object - Represents more complex data structures (e.g., arrays, objects).

Functions

Functions are reusable blocks of code that perform specific tasks. Here's an example of a simple function in JavaScript:

function greet(name) {
  return 'Hello, ' + name + '!';
}

greet('John'); // Output: 'Hello, John!'

Loops

Loops are used to repeat a block of code multiple times. The most common loops in JavaScript are:

  • for loop - Used to repeat code a specific number of times.
  • while loop - Used to repeat code as long as a condition is true.

Example of a `for` loop:

for (let i = 0; i < 5; i++) {
  console.log(i);
}
